STACK 📄
index.html
Single-file app — HTML + CSS + Vanilla JS
POST /api/chat  ·  body: { message: "student input" }
Server Layer
01
Serves index.html to the student's browser on GET /
02
Receives the student's message via POST /api/chat and parses the JSON body
03
Injects the BC CourseFinder™ system prompt — defines the Jarvis persona, language, scope, and ethical guardrails
04
Forwards the request to the Gemini API — API key is stored server-side and never exposed to the browser
05
Receives the AI response, wraps it in JSON, and returns it to the browser with Content-Type: application/json
HTTPS POST generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
AI Layer
MODEL 🤖
Gemini 2.0 Flash
Google's fast, capable large language model for conversational AI
PROMPT 📋
System Prompt
Shapes persona, tone, scope boundaries & ethical guardrails
OUTPUT 📝
Response Generation
Returns short, conversational, student-friendly guidance
SECURE 🔒
API Key Auth
Bearer token stored server-side — never exposed to client
